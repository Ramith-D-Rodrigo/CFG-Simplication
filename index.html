Context Free Grammar Simplification

The following are the symbols that are defined by default:
Start Symbol = S
Empty String = e
Enter the Non Terminal Symbols (Must be Upper Case) (Start Symbol is not needed) separated by comma (,)
Ex: A,B,C
Enter the Terminal Symbols (Must be Lower Case) (Empty String is not needed) separated by comma (,)
Ex: a,b,c,d
Click the button (+) to add the production rules.
Enter the production rules in the following format:
A -> aB | bC
where A is a Non Terminal Symbol, a and b are Terminal Symbols and B and C are Non Terminal Symbols. "|" is used to separate the production rules.
